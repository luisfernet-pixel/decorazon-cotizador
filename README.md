# DecoraZon · Versión 3 completa

Esta versión incluye:
- PDF limpio desde la pestaña Cotización
- Recursos: crear, editar, eliminar
- Ítems: crear, editar, eliminar
- Subítems: crear, editar, eliminar
- Historial: abrir, duplicar, eliminar
- Cotizaciones: crear, editar y guardar cambios

## Pasos
1. Copia `.env.example` a `.env.local`
2. Completa tus variables de Supabase
3. Ejecuta:

```bash
npm install
npm run dev
```

4. Para Vercel, agrega:
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
